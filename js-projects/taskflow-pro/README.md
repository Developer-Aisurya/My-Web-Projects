# TaskFlow Pro 🚀

A high-performance, minimalist task management dashboard designed for modern workflows. This project focuses on clean UI/UX principles, persistent state management, and real-time user analytics.

**[✨ Live Demo Link](https://developer-aisurya.github.io/My-Web-Projects/js-projects/taskflow-pro/index.html)**

## 📖 Project Summary
TaskFlow Pro is a professional-grade productivity tool that elevates the standard "to-do list" into a sleek, dashboard-style experience. Built with an Apple-inspired minimalist aesthetic, it provides users with a distraction-free environment to manage their daily objectives while tracking their completion progress in real-time.

## 🛠️ Technical Highlights
- **State Persistence:** Integrated with Browser `LocalStorage` to ensure tasks and theme preferences (Light/Dark mode) remain intact after refreshing or closing the browser.
- **Dynamic Progress Analytics:** Features a custom-built progress engine that calculates and visualizes completion percentages instantly.
- **Modern Architecture:** Uses ES6+ features, including `crypto.randomUUID()` for unique task identification and CSS Custom Properties for seamless theme switching.
- **Responsive & Accessible:** Fully responsive design that adapts to mobile, tablet, and desktop screens.

## 🌟 Key Features
- **Smart Filtering:** Categorize tasks into 'All', 'Pending', and 'Completed' with zero-latency transitions.
- **Advanced UI:** Glassmorphism accents, Inter typography, and smooth CSS animations.
- **Search Logic:** Real-time search functionality to find specific tasks within a large list.
- **Progress Tracking:** A visual progress bar to motivate users by showing their daily achievements.
